Replication files for "Tuning Parameter-Free Nonparametric Density Estimation from Tabulated Summary Data" by Lee, Sasaki, Toda, & Wang

[Explanation of folders]
- IRS statistics of income (SOI):	contains raw and cleaned income data
- MEtab:				contains main files for analysis
- PikettyPI:				contains files that implement Piketty's Pareto interpolation method (appendix material)
- Simulation:				contains files used in simulation study
